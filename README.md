SpeakSwift
==========

SpeakSwift, an iOS app written in Swift, Apple's new programming language. This app makes your iPhone/iPad speak in 35+ languages using AVSpeechSynthesizer, AVSpeechSynthesisVoice and AVSpeechUtterance.
